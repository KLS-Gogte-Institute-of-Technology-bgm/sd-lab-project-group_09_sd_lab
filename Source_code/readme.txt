Please follow these steps to Run this project :-

1. Download XAMPP

2. "TEXT EDITOR" NOTEPAD++/ATOM/SUBLIME TEXT/ ETC.

3. "Recruitment_Management_System"

4. Download the zip file

5. Extract the file and copy "Recruitment_Management_System" folder

6. Paste inside root directory (where you install xammp local "xampp/htdocs") 

7. Open PHPMyAdmin (http://localhost/phpmyadmin)

8. Create a database with name recruitment_db

9. Import recruitment_db.sql file(from zip package in SQL file folder)

10. Run the script http://localhost/Recruitment_Management_System
